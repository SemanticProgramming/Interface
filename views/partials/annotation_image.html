Image navigation

  • Pan image: press and hold left mouse button
  • Zoom image: roll mouse wheel to zoom in and out

Keyboard

  • Del-key: delete selected shape (active object only)
  • Esc-key: cancel current drawing, change to 'Edit' mode

Draw freehand:

  1. Choose 'Line String' or 'Polygon' tool
  2. Hold 'Shift'-key while drawing

Edit shapes:

  1. Select 'Edit' mode (Esc-key)
  2. Mark shape by clicking into it
  3. Add nodes by click and drag on line between existing nodes
  4. Delete node by clicing on it
  5. Move node by cklick and hold mous button while dragging the node
