General information:

Almost all interactive elements of the application provide context sensitive help text. Place your mouse cursor over an element for about a second to see the tool tip. The indicator on the upper right corner shows if Morph·D·Base is connected to the the middleware via the websocket. Without the connection the application is non functional.

Sign up:

Create a new account by clicking on “Log in or sign up” tab in the page header and fill out the form. After signing up your account is ready to use. Please be aware that we do not have a password reset function, so do not loose your password.

Login:

Log into Morph·D·Base using your email address and your password. After logging in you are redirected to the preliminary start page. You may either use the MyMDB drop down menu or one of the buttons on the left side to navigate to the desired function.

Demos:

  • MDB Specimen entry → Create a new MDB specimen entry:
    create a new specimen entry as described in the ontology. The middleware checks the input according to the description and stores the entry in the tuple store. Changes are automatically stored.

  • MDB Specimen entry → Show MDB entry:
    showing a sample specimen entry. The data is loaded from the tuple store and the interface is provided by the composition described in the ontology.

  • My MDB:
    show and edit your user entry

  • MDB Description Entry → Create a new MDB description entry:
    create a new description entry as described in the ontology. The middleware checks the input according to the description and stores the entry in the tuple store. Changes are automatically stored.

  • MDB Description entry → Show MDB entry:
    showing a sample description entry. The data is loaded from the tuple store and the interface is provided by the composition described in the ontology.

  • MDB Description tool:
    This part is currently under development. Feel free to explore the tool with the provided sample data.
